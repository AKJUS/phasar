Detailed Description

A simple custom allocator that is used to allocate, e.g., AbstractMemoryLocation for the IDEExtendedTaintAnalysis.

It can be used to allocate trivially destructible objects of variable size, e.g., using llvm::TrailingObjects, that should live as long as the allocator.
