Detailed Description

template<typename N, unsigned K>
class psr::CallStringCTX< N, K >

Stores a call-string context that can be used in interprocedural monotone analysis to achieve (limited) context sensitivity.

Template Parameters
NType of the call-string elements.
KMaximal length the call string can have.
