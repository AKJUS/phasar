Detailed Description

template<typename AnalysisDomainTy, typename Container = std::set<typename AnalysisDomainTy::d_t>>
class psr::FlowEdgeFunctionCache< AnalysisDomainTy, Container >

This class caches flow and edge functions to avoid their reconstruction. When a flow or edge function must be applied to multiple times, a cached version is used if existend, otherwise a new one is created and inserted into the cache.
