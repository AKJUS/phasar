Detailed Description

template<typename IdT, typename ValueT, unsigned SmallSize = 0>
class psr::TypedVector< IdT, ValueT, SmallSize >

Wraps a llvm::SmallVector, allowing index-based access by IdT, instead of size_t.

Template Parameters
IdTThe index-type that should be used for operator[]. Must be losslessly convertible from and to size_t.
ValueTThe usual value_type of SmallVector.
SmallSizeThe size of the inline-storage of SmallVector (default: 0)
