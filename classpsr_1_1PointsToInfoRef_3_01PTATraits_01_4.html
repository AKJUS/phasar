Detailed Description

template<is_PointsToTraits_v PTATraits>
class psr::PointsToInfoRef< PTATraits >

A type-erased reference to any object implementing the PointsToInfoBase interface. Use this, if your analysis is not tied to a specific points-to info implementation.

This is a non-owning reference similar to std::string_view and llvm::ArrayRef. Pass values of this type by value.
