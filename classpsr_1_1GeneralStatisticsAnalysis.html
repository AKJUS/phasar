Detailed Description

Computes general statistics for a Module.

This class uses the Module Pass Mechanism of LLVM to compute some statistics about a Module. This includes the number of

  • Function calls
  • Global variables
  • Basic blocks
  • Allocation sites
  • Call sites
  • Instructions
  • Pointers

and also a set of all allocated Types in that Module.

This pass does not modify the analyzed Module in any way!
