Detailed Description

template<typename V, typename N>
class psr::AliasInfoRef< V, N >

A type-erased reference to any object implementing the IsAliasInfo interface. Use this, if your analysis is not tied to a specific alias info implementation.

This is a non-owning reference similar to std::string_view and llvm::ArrayRef. Pass values of this type by value.

Example:

LLVMAliasSet ASet(...);
LLVMAliasInfoRef AA = &ASet;
