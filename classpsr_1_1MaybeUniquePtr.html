Detailed Description

template<typename T, bool RequireAlignment = false>
class psr::MaybeUniquePtr< T, RequireAlignment >

A smart-pointer, similar to std::unique_ptr, that can optionally own an object.

Template Parameters
TThe pointee type
RequireAlignmentIf true, the datastructure only works if alignof(T) > 1 holds. Enables incomplete T types
