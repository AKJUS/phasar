Detailed Description

template<typename V, typename N>
class psr::AliasIteratorRef< V, N >

A type-erased reference to any object implementing the IsAliasIterator interface. Use this, if your alias-aware analysis just needs a minimal interface to work with aliases and does not require the versatility of AliasInfoRef.

This is a non-owning reference similar to std::string_view and llvm::ArrayRef. Pass values of this type by value.

Example:

LLVMAliasSet ASet(...);
LLVMAliasIteratorRef AA = &ASet;
