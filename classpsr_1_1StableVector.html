Detailed Description

template<typename T, typename Allocator = std::allocator<T>>
class psr::StableVector< T, Allocator >

A special-purpose container designed for providing the almost complete interface from std::vector, but with the guarantee that references to stored elements are not invalidated by insertion or deletion of other elements.

This container can be used as memory owner for objects of any size providing the allocator properties of std::pmr::monotonoc_buffer_resource but with the convenience of a standard container.

NOTE: This container performs slightly to very much better than std::deque depending on the task.
