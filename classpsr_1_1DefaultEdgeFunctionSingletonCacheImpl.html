Detailed Description

template<typename EdgeFunctionTy, typename L>
class psr::DefaultEdgeFunctionSingletonCacheImpl< EdgeFunctionTy, L >

Default implementation of EdgeFunctionSingletonCache.

For an edge function EdgeFunctionTy to be cached, it must be hashable, i.e. it must implement the friend- or nonmember function llvm::hash_code hash_value(const EdgeFunctionTy&).

This cache is not thread-safe.
