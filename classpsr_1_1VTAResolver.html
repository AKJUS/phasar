Detailed Description

A Resolver that uses a variant of the Variable Type Analysis to resolver indirect calls.

Uses debug-information to achieve better results with C++ virtual calls. Uses alias-information as fallback mechanism for when types don't help or are not found, e.g., to resolve function-pointer calls.

Requires a base-call-graph or at least a base-resolver to resolve indirect calls while constructing the type-assignment graph.
