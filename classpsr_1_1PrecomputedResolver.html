Detailed Description

A Resolver that uses a pre-computed call-graph to resolve indirect calls.

Note
We eventually may want the LLVMBasedCallGraph to be a Resolver. This requires the concept of resolvers to generalize beyond LLVM. See https://github.com/fabianbs96/phasar/tree/f-ResolverCombinators for reference
