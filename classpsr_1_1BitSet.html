Detailed Description

template<typename IdT, typename BitVectorTy = llvm::BitVector>
class psr::BitSet< IdT, BitVectorTy >

A set-type that can compactly store sets of sequential integer-like types.

Use this type for sequential (unsigned) integers and ids that can convert from and to uint32_t.

Template Parameters
IdTThe type of elements to store in this set. Must be losslessly convertible from and to uint32_t.
BitVectorTyThe underlying bit-vector to use. Must be either llvm::BitVector or llvm::SmallBitVector.
