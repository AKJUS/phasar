Detailed Description

template<typename N, typename F>
class psr::CallGraph< N, F >

An explicit graph-representation of a call-graph. Only represents the data, not the call-graph analysis that creates it.

This type is immutable. To incrementally build it from your call-graph analysis, use the CallGraphBuilder
