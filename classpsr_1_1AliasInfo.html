Detailed Description

template<typename V, typename N>
class psr::AliasInfo< V, N >

Similar to AliasInfoRef, but exclusively owns the held reference. Use this, if you need to decide dynamically, which alias info implementation to use.

Implicitly convertible to AliasInfoRef.

Example:

AliasInfo AA = std::make_unique<LLVMAliasSet>(...);
