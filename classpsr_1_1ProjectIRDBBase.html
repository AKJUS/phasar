Detailed Description

template<typename Derived>
class psr::ProjectIRDBBase< Derived >

This class owns the IR code of the target being analyzed and some very important information associated with the IR. When an object of this class is destroyed it will clean up all IR related stuff that is stored in it.

m_t - module type f_t - function type n_t - instruction type g_t - global variable type
