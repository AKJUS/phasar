55/// A smart-pointer, similar to std::unique_ptr, that can optionally own an
56/// object.
57///
58/// \tparam T The pointee type
59/// \tparam RequireAlignment If true, the datastructure only works if
60/// alignof(T) > 1 holds. Enables incomplete T types
